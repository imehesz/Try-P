Let's play with strings a little. How about creating a simple poem generator. 

$words = array( 'Dracula', 'flies', 'bat', 'dawn', 'comes', 'night', 'child', 'faith', 'love', 'hearth', 'dog', 'dusk', 'from', 'the', 'egg', 'plastic', 'wings', 'flower', 'eat', 'drink', 'foolish', 'horse', 'dog', 'cat', 'plays', 'write', 'read', 'secret', 'star', 'darkness', 'young', 'today', 'yesterday', 'tomorrow', 'hear', 'funny', 'tree', 'under', 'honey', 'then', 'when', 'if', 'is' );

$number_of_lines = 4;
$number_of_words = 5;
$number_of_blocks = 3;

for( $block_cnt=0; $block_cnt < $number_of_blocks; $block_cnt++ ){
    for( $line_cnt=0; $line_cnt < $number_of_lines; $line_cnt++ ){
        for( $word_cnt=0; $word_cnt < $number_of_words; $word_cnt++ ){
            echo $words[ rand( 0, sizeof( $words )-1 ) ] . ' ';
        }
        echo '<br />';
    }
    echo '<br />';
}

So at first we fill an array with words, then we set some variables: how many blocks we would like to see, how many lines within a block and how many words within a line. Then we simply loop through each of these, and we print out a random word. When we hit the 5th word, we add a line break, when we finish a block, we add another line break for more space.

As an excersize you could add random punctuations here and there to make the poem more theatrical and maybe capitalize every blocks' first word's first letter.
Hint: use ucfirst( $word ) to make the first letter upper-case.

For extra kudos you can make sure that you only use a word once ;)
