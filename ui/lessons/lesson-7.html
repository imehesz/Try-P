Strings

A string is traditionally a sequence of characters, either as a literal constant or as some kind of variable. You already met a string called "Bela" in lesson 4. 

// in PHP you can set strings like this
$first_name = 'Bela';
$last_name = "Lugosi"; // note the double quotes!

String Functions

You can do a bunch of things with strings (other than play the violin): 

// you can chain them
echo $first_name . ' ' . $last_name; // Bela Lugosi

// use special function for special effects like ...
echo strrev( strtoupper( $first_name ) ); // ALEB

You can read more at String Functions.
