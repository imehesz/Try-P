An integer is a number of the set ℤ = {..., -2, -1, 0, 1, 2, ...}. Integers can be specified in decimal (base 10), hexadecimal (base 16), octal (base 8) or binary (base 2) notation, optionally preceded by a sign (- or +). 

$a = 1234;
$a = -123;
$a = 0123; // octal number!
$a = 0x1A; // hexadecimal number!

Floating Doubles

Floating point numbers (also known as "floats", "doubles", or "real numbers") can be specified using any of the following syntaxes: 

$a = 1.234;
$b = 1.2e3;
$c = 7E-10;

You can read more at Integers and Floating Point Numbers.
