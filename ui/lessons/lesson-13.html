Alright, with the knowledge we have so far, we should be able to do some cool stuff. Let's write a program that will select a secret number between 0 and 100 and find out what the number is programatically.

A very simple solution would be to go through the numbers from 0 to 100, but it would not be effecient. There must be a better way ... and there is. 

$secret_number = rand(0,100); // picking a secret number
$tip = rand( 0,100 ); // we set the first tip
$step = 1; // we wanna count how many steps it will take

// we run this code block until we find the right number 
while( $tip != $secret_number ){
    if( $tip < $secret_number && $step < 100 )
    {
		$tip += round( (100-$tip)/2 );
    }
    else
    {
		$tip -= round( (100-$tip)/2 );
    }
    //$tip = rand( 0,100 );
    $step++;
}

echo "YES! It took $step steps to find " . 
     "the secret number: $secret_number!";
