Alright, with the knowledge we have so far, we should be able to do some cool stuff. For example, let's add the numbers together from 1 to 100. (1+2+3+4+5+6 ... etc) 

$numbers = range(1,100); // $numbers, an array filled with 1,2,...,99,100
$sum = 0;

foreach( $numbers as $number ){
    $sum += $number; // same as $sum = $sum + $number
}

echo $sum;

You can also add an extra parameter to the range() function to skip over to every second, third, fourth etc number. So it'd look like this range(1,100,2). Give it a try.
