Functions

You've been using PHP functions left and right so you might be wondering ...
"Can I write my own functions!?" - in short - YES, YOU CAN!

Hopefully you remember when in Lesson 7 we used two functions to reverse and make a string uppercase. It looked something like this: echo strrev( strtoupper( 'string' ) );. If we would like to do this many times in different parts of our code, it would be more convenient to just call one function instead ... 

// declaring a new function
function revupper( $str ){
    // we create a new value with the change string
    $return_value = strrev( strtoupper( $str ) );
    return $return_value; // we return the return_value ;)
} 

// or even better
function better_revupper( $str ){
    return strrev( strtoupper( $str ) );
}

// now let's test
echo revupper( 'Bela' ); // ALEB
echo better_revupper( 'Lugosi' ); // ISOGUL
