Object

Many many years ago developers felt like programming languages should feel more natural to humans and reflect "real life" better. So they came up with the term Object Oriented Programming or OOP. What is an object? Cars, books, animals, flowers are all objects. Simple, isn't it?

There are at least a zillion books there about OOP and it would be impossible to explain everything about objects here. In a tiny nutshell ... 

// defining classes ...
class Vehicle{
	public $name;
	public $maker = 'Robots';
	public $can_fly = false;
	public $max_speed = 0;
}

class Car extends Vehicle{
	public $max_speed = 200;
}

class Spaceship extends Vehicle{
	public $maker = 'NASA';
	public $can_fly = true;
	public $max_speed = 43200;
}

// ... and our objects
$voyager = new Spaceship();
$focus = new Car();
$focus->maker = 'Ford'; // updating properties

// accessing properties
echo $voyager->max_speed; // 43200
echo $focus->maker; // Ford
