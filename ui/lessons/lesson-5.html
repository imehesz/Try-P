Booleans

Boolean is the simpliest type of variable. It expresses a truth value. It can be either TRUE or FALSE.

How to use?

$foo = true;

In PHP the following values interpreted as FALSE:

  • boolean false
  • integer 0
  • float 0.0
  • empty string "" and string zero "0"
  • an array with 0 elements
  • a special type, NULL

You can read more at Boolean Type.
