Arrays I.

An array type is a data type that is meant to describe a collection of elements (values or variables), each selected by one or more indices (identifying keys) that can be computed at run time by the program. You're probably thinking ... meh? 

// set an array
$arr = array( 'first_name' => 'Bela', 'last_name' => 'Lugosi' );
$arr2 = array( 11,22,33,44 ); // keys will be: 0,1,2,3

// print_r() displays information about a variable
print_r( $arr ); // Array ( [first_name] => Bela ... )
print_r( $arr2 ); // Array ( [0] => 11 [1] => 22 ... )

// access a value within an array
echo $arr['last_name']; // Lugosi
echo $arr2[2]; // 33
