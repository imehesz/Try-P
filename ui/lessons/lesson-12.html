Snoop-a-LOOP ... a

Loops are simply repeating code blocks a certain amount of time. 

// a for loop
for( $cnt=0; $cnt<5; $cnt++ )
{
  echo 'XO'; // XOXOXOXOXO
}

At the beginning of the FOR loop we have a declaration, $cnt=0; the middle tells you how many times it's going to execute, while $cnt is smaller than 5, at last we simply increase the $cnt counter by one with `++`. 

// a foreach loop
$poem = array( 'Listen to them...','children of the night.','What music they make!');
foreach( $poem as $line ){
    echo $line . '<br />';
}

// or
$employees = array(
    'Renfield' => 44,
    'John Harker' => 39,
    'Dr. Seward' => 57
);
foreach( $employees as $name => $age ){
    echo "Name:$name, Age: $age<br />"; 
    // note the usage of " quotes and variables within a string!
}

As you can see a foreach loop is good to loop through array elements. You could use a regular for loop for most of the cases, foreach is more convenient and easier to read. 

// a while loop
$cnt = 0;
while( $cnt < 10 ){
    echo $cnt;
    $cnt++;
} // we print the value of $cnt until it reaches 9
