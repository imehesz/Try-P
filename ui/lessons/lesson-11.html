If Then Else ... ElseIf

IF statements are very simple. Well ... most of the time. They check if a certain condition is true or false and execute a piece of code. Let's see an example: 

$a = 10;
$b = 99;

// if $a is larger than $b ...
if( $a > $b ){
  echo '$a is bigger!'; // say $a is bigger ...
}
else{
  echo '$b is bigger'; // otherwise say $b is bigger
}

It's good so far but what happens if the two numbers are equal? Yes, it would still say that `$b is bigger` because our condition only checks if $a is larger than $b. Let's fix it: 

$a = 10;
$b = 10;

// if $a is larger than $b ...
if( $a > $b ){
  echo '$a is bigger!'; // say $a is bigger ...
}
elseif( $b > $a ){
  echo '$b is bigger'; //  say $b is bigger
}
else{
  echo '$a equals $b'; // otherwise say they are equal
}
