Numbers

As you learned from the previous lesson how to print something to the screen, let's try to do something useful. For example add two numbers together.

echo 7+2; // 9

Of course other math related operations are working too like 3-1, 5/10 or 7*8. You can use more complex formulas as well. 

echo ( 1+2*(1+1)/4 ); // use the code editor to find out :P

Other Math functions

There are also a number of other functions you can use. For example to get the square root of a number you would use: 

echo sqrt(9); // 3

You can read more at PHP Math functions.
