Your first PHP code

If you have ever learned any programming languages, you are proabably familiar with Hello World!. So let's see how it's done with PHP. 

echo 'Hello World!';

When you embed PHP code into a website, you'll need to use the <?php ... ?> opening and closing tags. On this site we'll do it for you, so don't worry about it ;)

From this one line you can already learn a lot. The word echo is a command which tells the PHP interpreter what to do, usually followed by ([params]) like 'Hello World!' and close the whole line it with a semi-colon `;`.

You can read more at Your first PHP-enabled page.
